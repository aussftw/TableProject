## Usage

Install npm packages

```npm install```

Run app in development mode

```npm run dev```

Open ```http://localhost:3000 ``` to see app in browser.

### Technology Stack

- TypeScript
- React JS: Redux, Hooks;
- SSR: Next.js
- React styles & libraries: axios, styled-components,  see more in package.json;
